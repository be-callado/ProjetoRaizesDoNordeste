# ProjetoRaizesDoNordeste
Desenvolvimento de um site para trabalho de conclusão do curso de Análise e Desenvolvimento de Sistemas. - Trilha Front-End

Objetivo: simula um cenário real de mercado onde o aluno deve projetar uma solução
robusta para uma rede de lanchonetes em expansão. O foco está na compreensão do negócio, na
tomada de decisão técnica e na entrega de uma interface que suporte múltiplos canais de
atendimento (App, Totem e Web).
